## 数据库设计

### 1. 产品表 (products)

```sql
CREATE TABLE products (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    sku             VARCHAR(50) UNIQUE,           -- 产品编号
    name            VARCHAR(255) NOT NULL,        -- 产品名称
    subtitle        VARCHAR(255),                 -- 副标题
    description     TEXT,                          -- 产品描述
    price           DECIMAL(10,2),                 -- 价格
    currency        VARCHAR(3) DEFAULT 'USD',     -- 货币单位
    
    -- 图片
    main_image      VARCHAR(500),                 -- 主图URL
    dimension_image VARCHAR(500),                  -- 尺寸图URL
    
    -- 关联
    category_id     INT,                          -- 分类ID
    template_id     INT,                          -- 模板ID（可按分类自动关联）
    
    -- 状态
    status          ENUM('draft','active','archived') DEFAULT 'draft',
    
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 2. 分类表 (categories)

```sql
CREATE TABLE categories (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    name            VARCHAR(100) NOT NULL,        -- 分类名称
    slug            VARCHAR(100) UNIQUE,          -- URL别名
    parent_id       INT DEFAULT NULL,             -- 父分类ID（支持多级）
    default_template_id INT,                      -- 默认模板ID
    
    FOREIGN KEY (parent_id) REFERENCES categories(id)
);
```

### 3. 模板表 (templates)

```sql
CREATE TABLE templates (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    name            VARCHAR(100) NOT NULL,        -- 模板名称
    slug            VARCHAR(100) UNIQUE,          -- 模板标识
    category_id    INT,                          -- 适用分类（NULL=通用）
    
    -- 主题配置
    primary_color   VARCHAR(7) DEFAULT '#1e40af',  -- 主色调
    accent_color    VARCHAR(7) DEFAULT '#3b82f6',  -- 强调色
    
    -- 模板文件路径
    template_path   VARCHAR(255),                 -- HTML模板文件路径
    
    -- 元数据
    description     TEXT,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4. 产品快速特性 (product_quick_features)

```sql
CREATE TABLE product_quick_features (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    product_id      INT NOT NULL,
    
    icon_svg        TEXT,                         -- SVG图标代码
    text_line1      VARCHAR(100),                 -- 第一行文字
    text_line2      VARCHAR(100),                 -- 第二行文字
    sort_order      INT DEFAULT 0,                -- 排序
    
    FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
);
```

### 5. 产品功能卡片 (product_feature_cards)

```sql
CREATE TABLE product_feature_cards (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    product_id      INT NOT NULL,
    
    icon_emoji      VARCHAR(10),                  -- 图标（emoji或SVG路径）
    title           VARCHAR(100) NOT NULL,        -- 卡片标题
    description     TEXT,                         -- 卡片描述
    sort_order      INT DEFAULT 0,
    
    FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
);
```

### 6. 产品规格参数 (product_specifications)

```sql
CREATE TABLE product_specifications (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    product_id      INT NOT NULL,
    
    spec_label      VARCHAR(100) NOT NULL,        -- 规格名称（如"材质"）
    spec_value      VARCHAR(255) NOT NULL,       -- 规格值（如"钢铁"）
    sort_order      INT DEFAULT 0,
    
    FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
);
```

### 7. 产品标签 (tags)

```sql
CREATE TABLE tags (
    id              INT PRIMARY KEY AUTO_INCREMENT,
    name            VARCHAR(50) NOT NULL,
    slug            VARCHAR(50) UNIQUE
);

CREATE TABLE product_tags (
    product_id      INT,
    tag_id          INT,
    PRIMARY KEY (product_id, tag_id),
    FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE,
    FOREIGN KEY (tag_id) REFERENCES tags(id) ON DELETE CASCADE
);
```

## AI 生成流程

```
┌────────────────────────────────────────────────────────────────┐
│                     AI 产品页面生成流程                          │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  1️⃣ 读取产品数据                                                │
│     └─▶ products + 关联表（特性/卡片/规格/标签）                  │
│                                                                │
│  2️⃣ 确定模板                                                   │
│     └─▶ 按 category_id 或 template_id 选择模板                  │
│                                                                │
│  3️⃣ 数据填充                                                   │
│     └─▶ 将数据映射到模板占位符                                  │
│                                                                │
│  4️⃣ 输出HTML                                                   │
│     └─▶ 生成完整的产品详情页                                    │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 技术栈推荐

| 层级 | 推荐方案 |
|------|---------|
| **前端** | 模板引擎 (Handlebars/Nunjucks) + 您的HTML模板 |
| **后端** | Node.js / Python Flask / PHP |
| **数据库** | MySQL / PostgreSQL / SQLite（轻量） |
| **AI集成** | OpenAI API / Claude API 生成描述文字 |
