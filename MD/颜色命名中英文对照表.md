## 七、命名规范建议 / Naming Convention

### 7.1 自然色系命名规则
```
[季节/状态] + [基础色]
例如：夏季绿、冬季绿、秋黄、枯黄、枯棕色
```

### 7.2 基础色系命名规则
```
[深浅] + [基础色]
例如：深绿、浅棕、深棕、墨绿
```

### 7.3 英文命名规则
```
[修饰词] + [基础色]
例如：Emerald Green, Dark Olive, Withered Brown, Autumn Yellow
```

## 八、使用说明 / Usage Notes

1. **新增产品时**：请先查阅此表，保持颜色命名一致性
2. **中英文对照**：确保中文和英文含义对应准确
3. **避免重复**：同一产品内避免使用相似颜色名称
4. **广告法审查**：颜色命名不涉及极限词，无需额外审查
