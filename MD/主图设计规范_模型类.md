# 模型类产品主图设计规范

## 一、SKU图设计要点

### 1.1 SKU图核心要求

SKU图是买家快速识别产品变体的关键，需要在有限空间内展示最多的有效信息。

```
┌─────────────────────────────────────────────────────────┐
│                    SKU图布局结构                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐     │
│   │SKU1 │ │SKU2 │ │SKU3 │ │SKU4 │ │SKU5 │ │SKU6 │     │
│   │     │ │     │ │     │ │     │ │     │ │     │     │
│   │     │ │     │ │     │ │     │ │     │ │     │     │
│   └─────┘ └─────┘ └─────┘ └─────┘ └─────┘ └─────┘     │
│                                                         │
│   ├── 产品图（白底/透明底）                               │
│   ├── 尺寸标注（cm/inch）                               │
│   ├── 颜色名称（中英文）                                 │
│   ├── SKU编码                                           │
│   └── 数量/比例标注                                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.2 SKU图信息层级

| 优先级 | 信息类型 | 字号建议 | 说明 |
|--------|----------|----------|------|
| **P1** | 产品图 | - | 占图60%以上，白底或透明底 |
| **P2** | 颜色区分 | 主标题14-16pt | 用色块或色条区分不同颜色 |
| **P3** | 尺寸规格 | 12-14pt | cm/inch双单位 |
| **P4** | SKU编码 | 10-12pt | 便于库存管理 |
| **P5** | 比例信息 | 12pt | 模型类产品特有参数 |

### 2.2 主图类型矩阵

| 图序 | 图片类型 | 占比建议 | 核心目的 | 设计要点 |
|------|----------|----------|----------|----------|
| **图1** | 白底产品图 | 必选 | 品牌专业感 | 纯白底/灰底，突出产品轮廓 |

### 2.3 第一张主图（白底核心图）

#### 设计要求
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│                    纯白/浅灰背景                         │
│                   (RGB: 255,255,255)                    │
│                                                         │
│                      ┌───────┐                          │
│                      │       │                          │
│                      │  产品  │ ← 占比60-70%            │
│                      │       │                          │
│                      └───────┘                          │
│                                                         │
│                      产品居中                            │
│                    无阴影/浅阴影                          │
│                    高清300dpi                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 规范要点
- 背景色：纯白 `#FFFFFF` 或浅灰 `#F5F5F5`
- 产品占比：60-70%，不低于50%
- 边缘留白：不少于图片宽度的5%
- 图片尺寸：800×800px以上，正方形最佳
- 无水印、无文字、无边框

## 三、主图优化 Checklist

### 3.1 视觉优化

| 检查项 | 要求 | 优先级 |
|--------|------|--------|
| 背景纯白 | 无渐变、无杂质 | ★★★ |
| 产品清晰 | 无模糊、无毛边 | ★★★ |
| 比例协调 | 产品占比60-70% | ★★★ |
| 光线均匀 | 无过曝、无暗角 | ★★ |
| 色彩准确 | 与实物一致 | ★★ |
| 构图平衡 | 产品居中偏上 | ★★ |

### 3.2 信息优化

| 检查项 | 要求 | 优先级 |
|--------|------|--------|
| 产品名称 | 首图底部或第二张 | ★★★ |
| 尺寸规格 | cm+inch双单位 | ★★ |
| SKU信息 | SKU图完整展示 | ★★ |
| 比例信息 | 模型类特有标注 | ★★ |
| 卖点文案 | 简洁有力 | ★ |
| 品牌标识 | 角标或水印 | ★ |

### 3.3 合规优化

| 检查项 | 要求 | 优先级 |
|--------|------|--------|
| 无侵权 | 原创素材 | ★★★ |
| 无违禁词 | 极限词禁用 | ★★★ |
| 无虚假宣传 | 数据真实 | ★★★ |
| 图片清晰度 | 800×800px+ | ★★ |

## 四、文案设计规范

### 4.1 产品名称结构

```
【标准格式】
品牌名 + 产品类型 + 核心特点 + 规格数量

【示例】
HO比例3D海绵灌木丛 仿真微型植物 8色可选

HO Scale 3D Sponge Shrub Clusters Miniature Foliage 8 Colors
```

### 4.2 卖点文案结构

```
【卖点公式】
痛点 + 解决方案 + 效果

【示例】
❌ 错误：路灯模型带灯
✓ 正确：预装LED灯 → 通电即亮 → 营造真实光影效果

❌ 错误：灌木丛装饰
✓ 正确：手工海绵树叶 → 自然不规则形态 → 模拟真实树篱效果
```

### 4.3 尺寸文案规范

```
【多规格标注】
尺寸：7cm×14cm (2.76"×5.51")
高度：1-3cm (0.4-1.2")

【多比例标注】
比例：1/75 | 1/100 | 1/150 | 1/200
Scale: 1:75 | 1:100 | 1:150 | 1:200
```

## 六、常见问题规避

### 6.1 主图常见问题

| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 背景杂乱 | 显得廉价 | 纯白/浅灰背景 |
| 产品过小 | 信息不明确 | 产品占60%+ |
| 光线不均 | 产品失真 | 柔光箱补光 |
| 色差严重 | 退货率上升 | 校色仪校色 |
| 文字过多 | 视觉疲劳 | 卖点精简 |

### 6.2 SKU图常见问题

| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 颜色无区分 | 无法选择 | 色块/色条标注 |
| 尺寸缺失 | 购买犹豫 | cm+inch双标注 |
| SKU编码乱 | 库存混乱 | 统一编码规则 |
| 信息拥挤 | 难以识别 | 合理留白 |
