# 图片目录

将产品图片放在此目录中，文件名格式：

- `{任意前缀}_main.jpg` - 主图
- `{任意前缀}_detail_1.jpg` - 详情图1
- `{任意前缀}_detail_2.jpg` - 详情图2
- `{任意前缀}_dimension.png` - 尺寸图
- `{任意前缀}_gallery_1.jpg` - 图库1

示例:
- main.jpg
- detail_view.jpg
- detail_1.jpg
- dimension.png

注意：文件名中需要包含 `_main_`、`_detail_`、`_dimension_` 等关键字来识别图片类型。
