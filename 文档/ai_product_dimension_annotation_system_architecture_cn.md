# 十、智能比例系统

# 推荐逻辑

用户只输入：

```text
真实宽度：120cm
```

系统自动：

- 计算像素比例
- 推导其它尺寸

---

# 公式

像素比例：

```text
真实宽度 / 像素宽度
```

例如：

```text
120cm / 600px
```

得到：

```text
0.2cm/px
```
