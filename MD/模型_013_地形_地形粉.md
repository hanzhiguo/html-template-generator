## 产品描述 / Product Description

### 中文描述

**产品名称：** 地形粉（6色地形装饰粉末）

**产品特点：**
地形粉采用静电植绒工艺，6.5×4.9厘米规格，包含6种不同颜色地形粉末。每种颜色模拟特定地貌特征：沙地、岩石、草地、泥土等。颗粒细腻均匀，粘附力强，可直接撒布在模型底座上，快速营造逼真的地形效果。适合沙盘模型、军事模型、建筑景观等多种微缩场景的地形制作。

### English Description

**Product Name:** Terrain Powder (6-Color Terrain Decoration Powder)

**Product Highlights:**
Terrain powder using static flocking process, 6.5×4.9cm size with 6 different color options. Each color simulates specific terrain features: sand, rock, grass, soil, etc. Fine uniform particles with strong adhesion, can be directly applied to model bases for instant realistic terrain effects. Perfect for diorama, military models, architectural landscapes and other miniature terrain building.
