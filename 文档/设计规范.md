## 八、动画与过渡

### 8.1 基础过渡

```tsx
// 颜色过渡
<button className="transition-colors">

// 全属性过渡
<div className="transition-all">

// 指定时长
<div className="transition-colors duration-200">
```

### 8.2 内置动画

```tsx
// 旋转加载
<div className="animate-spin">

// 脉冲
<div className="animate-pulse">

// 弹跳
<div className="animate-bounce">

// 淡入
<div className="animate-in fade-in duration-200">

// 缩放淡入
<div className="animate-in fade-in zoom-in-95 duration-200">
```

### 8.3 自定义动画

```css
/* 任务进度条动画 */
@keyframes task-strip-scroll {
  0% { background-position: 0 0; }
  100% { background-position: 240px 0; }
}

.task-running-animation::after {
  animation: task-strip-scroll 6s linear infinite;
}

/* 加载点动画 */
@keyframes loading-dot {
  0%, 80%, 100% { opacity: 0.3; }
  40% { opacity: 1; }
}
```

### 8.4 性能优化策略

```css
/* 默认禁用大部分过渡 */
*, *::before, *::after {
  transition-property: none !important;
}

/* 保留必要动画 */
.animate-spin, .animate-pulse {
  animation-play-state: running !important;
}

/* 窗口隐藏时暂停动画 */
body.window-hidden .animate-spin {
  animation-play-state: paused !important;
}
```

## 十五、最佳实践

1. **保持层级一致**：使用统一的背景层级系统
2. **品牌色点缀**：紫蓝渐变用于强调，不要过度使用
3. **半透明叠加**：`/50`、`/70`、`/80` 是常用透明度
4. **过渡性能**：优先使用 `transition-colors` 而非 `transition-all`
5. **响应式优先**：移动端优先，逐步增强
6. **语义化命名**：使用描述性的类名组合
