# 主图设计系统 — 颜色控件完整清单

> 本文档供专业设计师参考，列出主图编辑器所有可自定义颜色的元素及其对应控件 ID 和 CSS 选择器。

---

## 1. 文字层（Text Layer）

### 1.1 主标题（Main Title）

| 属性 | 控件 ID / key | 默认值 | 类型 |
|------|---------------|--------|------|
| 文字颜色 | `titleColor` | `#ffffff` | hex color |
| 描边颜色 | 硬编码 `rgba(0,0,0,0.8)` | — | — |
| 字体粗细 | `mainTitleWeight` | `bold` | `bold` / `normal` |
| 斜体 | `mainTitleItalic` | `false` | boolean |
| 字号 | `titleSize` | `48` | px |

### 1.2 副标题（Subtitle）

| 属性 | 控件 ID / key | 默认值 | 类型 |
|------|---------------|--------|------|
| 文字颜色 | `subtitleColor` | `#ffffff` | hex color |
| 描边颜色 | 硬编码 `rgba(0,0,0,0.8)` | — | — |
| 字体粗细 | `subTitleWeight` | `normal` | `bold` / `normal` |
| 斜体 | `subTitleItalic` | `false` | boolean |
| 字号 | `subtitleSize` | `28` | px |

### 1.3 文字效果

| 效果 | 属性 key | 默认值 | 描述 |
|------|----------|--------|------|
| 阴影 | `shadow` | `true` | 黑色阴影，`rgba(0,0,0,0.5)`, blur 10, offset 2 |
| 描边 | `stroke` | `false` | 黑色描边 `rgba(0,0,0,0.8)`, 主标题 3px, 副标题 2px |
| 背景 | `textBg` | `false` | 纯色背景块 |
| 毛玻璃 | `textBgGlass` | `false` | 磨砂玻璃效果 |

### 1.4 文字背景

| 属性 | 控件 ID / key | 默认值 | 类型 |
|------|---------------|--------|------|
| 背景颜色 | `textBgColor` | `rgba(0,0,0,0.5)` | rgba 含透明度 |
| 透明度 | 颜色 picker + 透明度滑条联动 | 50% | 0-100% |
| 圆角大小 | `textBgRadius` (滑条) | `16` | 0-40px |
| 毛玻璃强度 | `textBgGlassOpacity` | `0.25` | 5%-60% |

#### 毛玻璃渲染参数（不可调）

| 参数 | 值 |
|------|-----|
| 模糊半径 | 16px (box blur) |
| 玻璃底色 | 白色半透明 |
| 边框高光 | `rgba(255,255,255,0.4)`, 1.5px |
| 顶部光晕 | 渐变白到透明, 高度 35% |

---

## 2. 画布背景（Canvas Background）

| 属性 | 控件 ID / key | 默认值 | 类型 |
|------|---------------|--------|------|
| 背景颜色 | `bgColor` | `#f5f5f5` | hex color |
| 圆角 | `imageRadius` (滑条) | `0` | 0-40px |

---

## 3. LOGO 层（LOGO Layer）

| 属性 | 控件 ID / key | 默认值 | 类型 |
|------|---------------|--------|------|
| LOGO 颜色（仅 SVG） | `logoColor` | `#ffffff` | hex color |
| 大小 | `logoSize` (滑条) | `150` | 100-400px |
| 边距 | `logoMargin` (滑条) | `40` | 20-100px |
| 透明度 | `textLayerVisible` | `true` | 显示/隐藏 |

---

## 4. 尺寸标注层（Dimension Layer）

| 属性 | 控件 ID / key | 默认值 | 类型 |
|------|---------------|--------|------|
| 线条颜色 | `dimColor` | `#ff6b6b` | hex color |
| 线条粗细 | `dimLineW` (滑条) | `2` | 1-6px |
| 标注文字颜色 | `dimTextColor` | `#ff6b6b` | hex color |
| 文字大小 | `dimFontS` (滑条) | `16` | 12-36px |
| 文字背景 | `dimTextBg` (下拉) | `white` | white / none |
| 端点样式 | `dimEndStyle` (下拉) | `arrow` | arrow / dot / line |
