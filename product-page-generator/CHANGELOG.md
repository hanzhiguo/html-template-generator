# 主图模板生成器 - 功能更新日志

## 版本 v1.2.0 (2026-06-10)

### 🖌️ 新功能：涂抹移除

新增**涂抹移除**功能，支持用画笔涂抹覆盖图片上不需要的区域。

#### 功能特性

| 功能 | 说明 |
|------|------|
| **画笔涂抹** | 用鼠标在图片上涂抹，覆盖不需要的区域 |
| **自定义颜色** | 默认白色遮盖，可自定义任意颜色 |
| **画笔大小** | 5px - 200px 可调节 |
| **不透明度** | 10% - 100% 可调节 |
| **橡皮擦模式** | 擦除已涂抹的遮罩 |
| **撤销** | Ctrl+Z 撤销最后一笔 |
| **清空** | 一键清空所有涂抹 |
| **画笔光标** | 实时显示画笔大小预览圆圈 |
| **模板保存** | 涂抹数据随模板保存和加载 |

#### 渲染顺序

涂抹遮罩层在以下顺序中渲染：
```
图片 → 涂抹遮罩 → 文字 → LOGO → 标注
```

这确保涂抹不会遮住文字和LOGO。

#### AI 移除（预留）

已预留 AI 智能移除接口 `aiRemoveObject()`，后续可接入 AI 模型实现：
- 框选区域一键移除
- 智能背景填充
- 对象识别移除

#### 文件变更

| 文件 | 变更 |
|------|------|
| `main-image-template.html` | 新增涂抹 Tab、CSS样式、状态字段 |
| `main-image-brush-mask.js` | 新建：涂抹模块核心逻辑 |
| `main-image-render.js` | 渲染顺序调整 |
| `main-image-dimension.js` | 标注层互斥处理 |
| `main-image-template-manager.js` | 模板保存/加载集成 |
